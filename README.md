# MultiActions Mod
This is a mod for more personal use of what I would like to have in `VRChat`.

This mod adds the following things to the action menu:
* Teleport to players
* Teleporting:
  * Save points and name them
  * Find them in menu and teleport
* Quit game
* Toggle camera
* Respawn
* Open:
  * Avatars
  * Settings
  * Safety
